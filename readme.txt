Creates a widget showing a BigCommerce product.

== Description ==

Creates a widget showing a BigCommerce product. Users can choose from a dropdown of products and on the front optionally get an image, description, price, and buy button.

== Installation ==

1. Upload the `/bc-product-feature-widget/` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Appearance -> Widgets in the admin and place the widget in a sidebar

== Usage ==

Use the normal widget admin UI to drag the widget into your sidebar.

Choose a product.

Choose what features to render
